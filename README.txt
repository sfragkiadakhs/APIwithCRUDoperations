η άσκηση μου έτρεχε με intellij.

Εχω δημιουργήσει 2 interfaces για τα repo των books και ενα για τους users,
τα οποία κάνουν extend το JpaRepository και μας δίνουν μεθόδους με τη δυνατότητα πρόσβασης στη βάση,
 και 2 μοντέλα User και Book, για να επιδείξουν τα table που θα έχουμε στη βάση 
και μεθόδους για το τι θα αλλάξει σε αυτή,
ενω έχω εναν ApiController για την διαχείριση των request του κάθε path.


αφού έχω ανοίξει το mysql server, και τρέξω το πρόγραμμα
μπορώ να ανοίξω το postman Και να στείλω get,post,put και delete request.

για τα put χρειάζεται να διαλεξουμε raw και json format.
